# Installation

This document is docker installation on Ubuntu

## Prerequisites

If curl isn't installed, install it after updating your manager

~~~bash
apt-get update
apt-get install -y curl
~~~

## Get docker source

Get the latest Docker package.


~~~bash
curl -sSL https://get.docker.com/ | sh
~~~
