# Telegraf

## environment

Keyword     | Value                 | Description
----        | ----                  | ----
REPO        | https://dl.influxdata.com/telegraf/releases     | Telegraf Repository
VER         | 1.1.1     | Telegraf Version

# Installation

Download package

~~~bash
wget ${REPO}/telegraf_${VER}_amd64.deb
dpkg -i telegraf_${VER}_amd64.deb
~~~
