# Telegraf

## environment

Keyword     | Value                 | Description
----        | ----                  | ----
VER         | 1.1.1     | Telegraf Version

# Installation

Download package

~~~bash
wget https://dl.influxdata.com/telegraf/releases/telegraf_${VER}_amd64.deb
dpkg -i telegraf_${VER}_amd64.deb
~~~
